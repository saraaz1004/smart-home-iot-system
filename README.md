# Smart Home IoT System

Smart home IoT system using Arduino hardware, Python integration, sensor automation, and ThingSpeak cloud monitoring.

---

## Features

- Smart lighting system
- Temperature monitoring
- Motion detection
- Garage door automation
- Ventilation control
- Emergency mode
- Sensor automation
- ThingSpeak cloud integration
- Real Arduino hardware communication

---

## Hardware Components

- Arduino
- LM35 temperature sensor
- PIR motion sensor
- LDR light sensor
- Ultrasonic sensor
- Servo motor
- LEDs
- Relay modules

---

## Technologies Used

- Arduino C/C++
- Python
- ThingSpeak
- Embedded Systems
- IoT Automation

---

## Project Structure

```bash
arduinoKitCode.ino
smart_home.py
screenshots/
```

---

## Functionality

The system automatically controls lighting, ventilation, garage access, and motion detection using sensor data and Arduino automation logic.

Sensor data can also be monitored through ThingSpeak cloud integration.
